# AuroraView 使用说明（用户版）

本文面向终端用户，介绍 AuroraView 的基本操作、主要功能和常用快捷键。  
当前版本：1.0.2（修图工作流与外部打开增强）。

---

## 1. 快速开始

- 启动应用后，会进入主浏览界面（左侧为目录，右侧为缩略图网格）。
- 打开图片目录：
  - 点击顶部工具栏左侧的「文件夹」图标；
  - 或使用快捷键：`⌘/Ctrl + O`；
  - 选择包含图片的目录后，右侧会加载该目录下的所有图片缩略图。
- 浏览缩略图：
  - 单击：选中/多选图片；
  - 双击：打开大图预览；
  - 右键：打开上下文菜单（外部应用打开、在 Finder 中显示等）。

---

## 2. 目录与侧边栏

- 左侧侧边栏包含以下分组：
  - **收藏目录**：常用工作目录，支持拖拽和手动添加；
  - **星级图片**：所有已评级图片的聚合视图；
  - **外接存储**（仅 macOS）：自动列出已挂载的 U 盘、移动硬盘等；
  - **系统目录**：常见的系统图片目录（如「图片」等）；
  - **自定义目录**：手动添加的任意路径。
- 常用操作：
  - 点击目录名称：切换当前浏览目录；
  - 点击「＋」按钮：向对应分组添加新目录；
  - 右键图片 →「复制到/移动到目录」：可与侧边栏组合使用，快速整理图片。

---

## 3. 缩略图视图与布局

- 布局切换：
  - 顶部中间的「网格图标」下拉菜单可切换：
    - 标准视图：缩略图较大，信息更完整；
    - 紧凑视图：缩略图更密集，适合浏览大量图片。
- 缩略图交互：
  - 单击：选中图片；
  - `⌘/Ctrl + 单击`：多选图片；
  - `Shift + 单击`：选中连续范围；
  - 点击空白区域：清空当前选中；
  - 右键：打开上下文菜单。
- 排序：
  - 使用排序下拉菜单，可以按名称、时间、大小、文件类型等进行排序。

---

## 4. 修图状态：已修 / 未修（PSD + RAW 工作流）

AuroraView 为同名 RAW/PSD 工作流设计了简单的“已修/未修”状态：

- 状态模型：
  - 以 PSD 的虚拟标签为唯一“已修”来源，不修改真实文件名；
  - 在同一目录下，将 `photo.dng / photo.jpg / photo.psd` 等同名文件视为一组；
  - 组内有 PSD 且该 PSD 被标记为已修 → 整组视为「已修」；
  - 否则，若组内存在 RAW/DNG 或 PSD → 整组视为「未修」。
- 缩略图角标：
  - 右上角绿色角标「已修」：说明该组存在已修 PSD；
  - 深色角标「未修」：说明该组尚未完成修图（仅 RAW/PSD 而无已修 PSD）；
  - 角标不会改变真实文件名，只是 UI 提示。
- PSD 标签按钮：
  - 在 PSD 缩略图的文件名之前会显示一个标签按钮：
    - 已修：`[已修]`（高亮，点击可改为未修）；
    - 未修：`标记为已修`（点击即可将该 PSD 标记为已修）。
- PSD 标记管理面板：
  - 通过菜单栏 `文件 → PSD 标记管理…` 打开；
  - 会列出当前目录中的所有「待标记」 PSD 分组，方便批量审阅和一键标记。

## 5. 修图筛选（已修 / 未修 / 全部）

- 顶部工具栏提供「修图筛选」下拉按钮（图标为文档/PSD 样式）：
  - 全部：显示所有图片；
  - 已修：仅显示被判定为「已修」的分组；
  - 未修：仅显示被判定为「未修」的分组。
- 筛选按分组生效：
  - 一旦某个分组被选中（例如 `photo` 为已修分组），该分组下的所有格式（PSD、RAW、JPG 等）都会一起显示；
  - 方便对比 RAW 与最终 PSD 的效果。
- 目录切换时：
  - 修图筛选会自动重置为「全部」，避免上一目录的筛选影响新目录。

## 6. 外部应用打开 & 打开方式绑定

- 缩略图右键菜单：
  - 打开（系统默认应用）；
  - 在 Finder 中显示；
  - 使用已配置应用打开（.psd / .dng 等）；
  - 选择应用并设为 .扩展名 默认打开方式…
- 打开方式绑定：
  - 首次为某种扩展名选择应用后，AuroraView 会将「扩展名 → 应用路径」保存在用户配置中；
  - 之后右键同类文件可直接使用该应用打开，例如：
    - `psd → Photoshop`；
    - `jpg → 预览`；
    - `dng/raw → 预览或其他专业软件`。
- 大图视图中的外部打开按钮：
  - 在大图模式顶部工具栏，可看到一个「外部应用打开」按钮；
  - 行为与右键菜单一致：优先使用已绑定应用，否则提示选择并绑定。

## 7. 常用操作与快捷键一览

### 7.1 文件与目录

- 打开目录：`⌘/Ctrl + O`
- 刷新当前目录：`⌘/Ctrl + R`
- 删除选中图片：`Delete` 或 `Backspace`
- 复制选中图片路径：`⌘/Ctrl + C`
- 导出选中图片：`⌘/Ctrl + E`
- 复制到其他目录：`⌘/Ctrl + Shift + C`
- 移动到其他目录：`⌘/Ctrl + Shift + M`
- 全选：`⌘/Ctrl + A`

### 7.2 大图视图

- 打开大图：双击缩略图或在选中项上按 `Space`
- 关闭大图：`Esc` 或点击关闭按钮
- 前一张 / 下一张：左右方向键
- 放大 / 缩小：工具栏按钮（支持触控板缩放）
- 复位视图：双击画面 / 工具栏“适应窗口”按钮

### 7.3 其他

- 切换主题：菜单栏 `视图 → 主题 → 自动/浅色/深色`
- 开启/关闭开发者工具：菜单栏 `视图 → 切换开发者工具`

---

## 8. 自动更新与版本提示（简要）

- AuroraView 在联网时会自动检查 GitHub Releases 上的新版本：
  - 如发现更新，将提示「发现新版本」；
  - 用户可选择立即重启安装，或稍后再说。
- 更新安装包基于 `.zip + latest-mac.yml` 文件：
  - `.dmg` 安装包主要用于首次安装或手动下载；
  - 自动更新逻辑不会依赖 dmg。
