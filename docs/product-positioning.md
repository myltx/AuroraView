# 产品定位：macOS 现代化图片查看器

## 1. 核心定义
- **产品名称（代号）**：Photon Viewer
- **目标平台**：macOS（兼容 Windows / Linux，但 macOS 体验优先）
- **产品类型**：桌面端图片浏览与管理工具
- **设计理念**：结合 Looker / Eagle / Pixave / macOS Photos 的现代设计，与 2345 看图王的目录效率

## 2. 目标用户与场景
| 用户类型 | 典型需求 | 使用场景 |
| --- | --- | --- |
| 创意设计师 / 品牌团队 | 快速预览大量素材、收藏灵感 | 项目素材管理、Moodboard 制作 |
| 摄影师 / 摄影爱好者 | 批量检索、筛选拍摄作品 | 外拍回片、作品集整理 |
| UI/UX 设计师 | 对比不同版本、整理图标/界面稿 | 评审会、设计交付 |
| 普通用户 | 便捷浏览本地照片，替代系统预览 | 家庭照片管理、旅行记录 |

核心使用场景：
1. **快速浏览**：导入任意目录，瞬间生成缩略图、筛选目标图像
2. **轻量管理**：使用 Finder 式目录树、收藏夹、标签管理图片
3. **专业预览**：通过 Lightbox 查看原始画质，查看 EXIF / 元信息

## 3. 竞争分析
- **Looker / Eagle / Pixave**：高级功能丰富，但偏重数据库式管理，上手成本高
- **macOS Photos**：强社交/云相册属性，不适合设计师本地素材库
- **2345 看图王 / Windows 浏览器**：侧边栏操作高效，但缺乏 macOS 现代视觉与手势支持
- **Photon Viewer**：结合 Finder 风格信息结构 + Mac 原生动效 + 轻量文件管理能力

## 4. 产品价值主张
1. **原生体验**：全面支持 macOS 深浅色、手势、快捷键、Finder 集成
2. **极速缩略图浏览**：采用虚拟列表 + 缓存策略处理海量图像
3. **智能目录与收藏**：Finder Style 的侧栏 + 收藏拖拽 + 最近浏览
4. **专业级查看器**：Lightbox 中提供缩放、旋转、EXIF、幻灯等操作
5. **轻量素材管理**：多选、标签、拖拽、删除、导出、复制

## 5. 成功指标（初版）
| 指标 | 目标 |
| --- | --- |
| 启动时间 | ≤ 3 秒（第一次加载大目录除外） |
| 缩略图生成速度 | 首次生成 1000 张图 < 30s，二次打开即时显示 |
| 目录切换响应 | < 500ms（已有缩略图） |
| Lightbox 操作延迟 | < 50ms |
| 用户满意度 | 设计师群体正向反馈（≥ 80%） |

## 6. 不做范围
- 图像编辑（滤镜、裁剪）
- 云端同步 / 团队协同
- AI 标签、自动分类（可列入长期规划）
