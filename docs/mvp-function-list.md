# MVP 功能清单

| 模块 | Must | Should | Won't (本阶段不做) |
| --- | --- | --- | --- |
| **目录侧栏** | 读取本地磁盘、展示目录树、展开/折叠、图标与缩进 | 收藏夹（拖拽添加、持久化）、系统常用路径（桌面/下载） | 云端目录、标签导航 |
| **全局搜索** | 文件名搜索、类型过滤（PNG/JPG/SVG 等） | Spotlight 集成、最近搜索 | 内容识别、AI 推荐 |
| **缩略图区域** | 自适应网格、子目录 tile、Hover 显示信息 | 缩略图尺寸 slider、类型徽标、信息浮层 | 瀑布流布局、智能分组 |
| **缩略图性能** | 懒加载、缓存目录、基础虚拟列表 | 预生成缩略图（Sharp）、缓存管理界面 | 分布式缓存、云共享缓存 |
| **查看器（Lightbox）** | 双击/点击弹出、平滑过渡、上一张/下一张、放大/缩小、旋转/翻转 | 手势缩放、拖拽移动、EXIF 信息、幻灯播放、记忆上次位置 | 多窗口编辑、批注、滤镜 |
| **操作与管理** | Cmd/Shift 多选、删除（废纸篓）、复制路径、打开所在文件夹 | 拖拽移动、导出、复制到剪贴板、重命名 | 标签系统、版本历史、云协作 |
| **系统集成** | 深浅色自动切换、Finder 右键“使用此应用打开”、文件夹更新自动刷新 | 默认图片查看器注册、快捷键配置 | iCloud 同步、iOS 配套 App |
| **工程基础** | Electron + Vite + Vue3 + TS、IPC 定义、Preload 暴露 FS API、安全沙箱 | 自动更新、日志/崩溃分析 | 多语言、插件系统 |

> MVP 范围定位为「本地图片浏览与轻量管理」，确保侧栏 + 缩略图网格 + Lightbox + 基础文件操作完整可用。高级编辑、云同步、AI 功能放入后续阶段。
